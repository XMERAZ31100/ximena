# Sistema de Gestión

Sistema de gestión estático con control de versiones y roles de usuario desarrollado con React y TailwindCSS.

## Características

- **Login con autenticación**: Sistema de inicio de sesión con validación de credenciales
- **Control de roles**: Tres niveles de usuarios (superusuario, admin, usuario)
- **Dashboard interactivo**: Panel con estadísticas, gráficos y calendario
- **Diseño profesional**: Paleta de colores rosa pastel con efectos elegantes
- **Sidebar colapsable**: Menú lateral con navegación fluida
- **Topbar funcional**: Barra superior con notificaciones y menú de usuario
